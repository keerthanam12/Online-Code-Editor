## üöÄ Run the Project
**1.** Download and Extract the zip file. Make sure to install the packages "NodeJS" and "NPM" and "React" in your environment variables.

**2.** Go to the File Explorer. Open the Extracted zip file.

**3.** Then go to VSCode and open folder and open this file.

**4.** Then click New Terminal and type [cd Online-Code-Editor] and click enter. 

**5.** Run the following commands:

```
npm start
```
or
```
npm install
```

**6.** Open a web browser. (http://localhost:8000/ or http://127.0.0.1:8000/)
